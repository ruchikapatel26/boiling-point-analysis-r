# Boiling Point Analysis using R

## Overview
This project performs exploratory data analysis (EDA) on a boiling point dataset using R.
It includes data cleaning, visualization, and statistical analysis.

## Dataset
- Original dataset: bp_data.csv
- Cleaned dataset: clean_data.csv
- Number of observations: 3462

## Tools Used
- R
- RStudio

## Analysis Performed
- Handling missing values
- Descriptive statistics (mean, median, standard deviation)
- Histogram of boiling points
- Scatter plot (ID vs Boiling Point)
- Correlation analysis
- Hypothesis testing (t-test)

## Visualizations
- Histogram of boiling points
- Scatter plot between ID and boiling point
